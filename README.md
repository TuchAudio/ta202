# TuchAudio 202
Simple, small stereo power amplifier

## Purpose
Simple, no-frills low power (~1.5 W RMS) stage, intended for connection thru the [501 Input Selector Board]() for the so called 'Bedroom Amp'; but could be used independetly as well.

## Specs

- **Power supply:** 12 V, asymmetrical, max. 400 mA
- **Gain:** 20x (+26 dB) with +46 dB option.
- **Input sensitivity:** 180 mV
- **Input impedance:** 10 kΩ; optional buffer for standalone operation.
- **Output level and impedance:** 18 Vpp / 8 Ω
